# 🛡️ Zhétical OSINT Case Tracker v2.0

Lien vers l'outil en ligne : [https://advanced-osint-case-jgpp.bolt.host](https://advanced-osint-case-jgpp.bolt.host)

## 🔍 À propos
Le **Zhétical OSINT Tracker** est un outil d'investigation numérique conçu pour aider les enquêteurs à structurer leurs recherches. Il permet de créer des graphes de liens interactifs entre différentes entités (Usernames, IP, Domaines, etc.) tout en garantissant une confidentialité totale.

## 🚀 Fonctionnalités Clés
* **Graphe de Liens Interactif** : Reliez vos indices avec des flèches pour visualiser le chemin de l'enquête.
* **Édition Rapide** : Double-cliquez sur n'importe quelle case pour prendre des notes instantanément.
* **OPSEC First** : Aucune donnée n'est envoyée sur un serveur. Tout est stocké localement dans votre navigateur.
* **Exports Professionnels** : Générez des rapports en PDF ou sauvegardez votre progression en JSON.
* **Intégration Ghostint-Tools** : Accès direct à mon toolkit d'outils OSINT via la barre latérale.

## 🛠️ Installation & Utilisation
Aucune installation requise. Cliquez simplement sur le lien ci-dessus pour lancer l'application dans votre navigateur.
